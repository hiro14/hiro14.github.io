0202年在 Pixel C 刷魔趣ROM

发表于 2020-03-28

前言

Pixel C 又开始电量尿崩了，放着不管一天多电就能跑光， timerfd 疯狂唤醒，就算清除 Google 数据都没用。那就一鼓作气 root 并且直接刷魔趣ROM上 Android 10 吧！

准备工作

阅读资料，下载资源

https://www.mokeedev.com/guide/getting_started.html

https://download.mokeedev.com/dragon.html

https://flash.mokeedev.com/ （其实没用，只是帮你完成一些步骤）

  • Magisk-v20.4.zip
  • MK100.0-dragon-202003280436-NIGHTLY.zip
  • open_gapps-arm64-10.0-nano-20200325.zip
  • twrp-3.3.1-2-dragon.img

把指定资源放进目标文件夹，再打开计算机的终端进入该文件夹。

刷机

解锁设备

进入设置-开发者选项-允许 oem 解锁

设备重启并显示 waiting for fastboot command….

按电源键确认，设备发出一声“滴”然后重启，显示“正在清空”（为什么是中文？）

有点慢……

为了确保正常，重启试试（没问题，恢复出厂了，跳过激活设置重新打开 usb 调试）

刷入 recovery

进入重启循环无法进入 recovery？

改成 fastboot boot .\twrp-3.3.1-2-dragon.img , 好了

选择 格式化 data 分区

在高级选项中选中 adb sideload，读条时刷入 mokee magisk opengapps

重启

卡在 Google 标志了？！

为了保险，重新线刷了一次，这次终于出现 mokee logo （后来发现，卡在 Google 标志，可能多重启几次会好，大概是 nightly 不够稳定）

妈的，这logo丑哭了

开机后设置

像一个正常 google 设备一样设置，从 Google 云拖回设备备份，更新应用……

Magisk Manager 有了，但是 safetynet 检查不通过，ctsProfile: false. 不行，这样装不了 Netflix.

安装 magiskhide props config 和 busybox 模块

顺便一说这个魔趣 ROM 自带终端，帮助我迅速解决 safetynet 问题，不用临时下一个 termux 了。直接在终端里操作：
