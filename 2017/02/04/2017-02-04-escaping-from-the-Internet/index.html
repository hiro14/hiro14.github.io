逃离互联网计划

发表于 2017-02-04

2月6日回学校工作。

今年春节过得也颇为清静，托 G20 的福，我浙放假晚，但是开学也晚呀。

然而放假关我屁事。

在家里休息了几天，人也懒了没把工作捡起来。重新配置了 Ubuntu 虚拟机环境，加了个 TensorFlow， 暂时把 MEAN 丢在一遍，毕竟和学校里的生产环境一点也搭不上边。重新配成了 LAMP。文章也没法看，因为家里的广电宽带 IP 一直跳来跳去，无法用学校 IP 访问期刊网站。更别提远程啦翻墙啦这类最基本的需求。 DigitalOcean 的 VPS 已经销毁了，因为没钱。

总之都是借口啦，春节期间除了修正了睡眠节律之外毫无建树。

前些天盯着自己的自动填充密码却怎么也想不起那一串星号背后的字母是啥了。心想脑子不靠谱自动备份其实也不靠谱。刚刚 GitLab 就崩了，原因是运维人员手残 + 备份全部失效。惨痛的教训也警醒了我，是时候减少自己对数据的依赖程度了。

接着前几天总结16年全年流水的余兴，我规划了一份逃离互联网的方案，当然不是完全逃离，基本的社交还是要保持的。断舍离也不能太彻底不是嘛。

那么先来列一列舍不得的内容吧，过把瘾之后再决定丢哪些东西。

保留内容：

  1. Twitter
  2. Telegram
  3. WeChat
  4. GitHub
  5. Inbox
  6. s1 和 reddit 账号

(to be continued)

妈蛋别偷懒，文章都已经发出去了。（我才发现现在 jekyll 已经可以显示来自未来的文章了，吃惊）

上面就是保留的东西，就是不需要动的部分。

精简内容：

  1. 云存储
  2. 自动存储密码
  3. QQ 多平台的聊天记录
  4. 各种电子日程表
  5. 各种订阅与收藏

以上内容每项都存放于多个位置，严重影响平时的工作管理。比如云存储包括 DropBox, OneDrive, Google Drive, iCloud Drive 等。密码有两个 Google 账号和一个 Firefox 账号。 QQ 就更不用说了，聊天记录在 PC， 手机和 iPod 上分别存在。

目前的进度是优化了各个云存储的自动同步设置避免彼此重复，删除本地冗余或无用文件，部分内容转移到 MEGA 网盘进行封存。密码还没着手行动，计划批量从浏览器设置中导出，希望能找到方法吧。当密码转移到本地之后会进行进一步的调整与断舍离行动，以消灭用不到的账号。聊天记录可以通过漫游找回，然后一个一个注销。以后尽量避免在多个位置使用 QQ， 并且少用 QQ。 订阅收藏是个老大难，然而无非是克服我的囤积癖罢了。 Pocket 是个好工具，可以重新用起来。日程就用 Trello 和 2Do 和奇妙清单打个组合拳好了，一个也不能少。

删除内容：

一切浪费钱和时间的账户和应用！

当然这句话不是说删掉花钱的账户，该花的钱还得花。只是有些不该浪费的时间却用在一些砸钱的内容上，根本无法取得进步。鉴于我目前的经济状况， 2017年可能无法自如地购入音乐和书籍以及网购各种产品。所以最重要的是封印淘宝（尤其是外卖）和 mora 账户。取下这两个账户的密码后就可能注销很长一段时间了。爱奇艺的账户不会续费，所以到五月为止吧。其他一切划水的账户，仅保留 s1 和 reddit。 cc98 的处理方案待定。

以上是能想到的部分。目前先做这么多吧。我定下这个计划也有其他的原因，这个我在自己的微博上也说过。

新年了，那就专门恭祝首页的各位身体健康，学业有成，工作顺利，家庭和睦，感情美满，珍爱生命，远离百度，有吃不完的美食，抽不尽的SSR，大把的自由时间和长久的独立人生，鸡年大吉吧。来微博七年了，有幸认识不少有趣有意思的人，你们让我见识到一个稍微广阔的世界，让我感受了每个人不同观念的相互碰撞，让我赞叹孤立个体的言语所发挥出的精神力量是多么强悍。如你所见，现在我们身处的环境允许的自由越来越少，开放的程度也在逐年降低。相同观念的人群联系越来越紧密，不同意见者之间的分歧越来越严重。也许有一天，对中国互联网最深的担忧即将成真。我从来不敢低估那一天来临的速度，我也决不可能承认一面墙存在的正当性。无论如何，任一阵营的人都应有平等发言的权利。因此，在还可以说话的时候，我不会放弃与真正的全世界沟通的桥梁。外面的世界并不精彩，充斥着更多的谎言与真实。唯一值得称道的一点就是你可以更大程度上借助自己的眼睛，而不是通过一部分人的引导与修饰，来确认一件事的好坏。现在的我们如果莫谈国事，民族和谐，吹吹水八个卦，写写闲言碎语，也许可以在这个上亿人的圈子中活得悠然自得。一年后呢？千万不要去适应。适者灭亡。

我对于自己有一天会在网络世界上突然消失这件事情是深信不疑的。虽然不如很多人那样混迹的够久，我还是在互联网上留下了难以割舍的印记。是时候回头看看，为自己做个整理了。
