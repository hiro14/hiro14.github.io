2017年10月动画选择

发表于 2017-10-28

七月半年番

  • Fate/Apocrypha

  • 舞动青春

十月新番

  • 宝石之国

  • 少女终末旅行

  • 新奇诺之旅

  • 泥鲸之子们在沙地上歌唱

  • 魔法使的新娘

Fate/Apocrypha

型月 神奇东出 搞笑 奇幻 神话 外传小说改

我对乱成一锅粥的圣杯战争没什么兴趣，身为非原作党，只是为了声优和部分从者角色而坚持下来的。新 OP 和 ED 很不错。

舞动青春

青春励志 竞技体育 漫改

虽然明显看出动画制作过程因为各种原因存在相当大的硬伤，但是有国人动作作画监督参与是作品质量的保险，而且后半程的剧情也越来越紧凑和有张力，千夏和多多良的互动也是当前的主要冲突，让人难以释怀。新 OP 和 ED 都非常好听。

宝石之国

奇幻 玄学 漫改

不管是3D还是剧情，都有种另类的惊艳。黑泽朋世和她演绎的磷叶石简直是合二为一，其他宝石的个性也很丰富，期待剧情层层推进。声优阵容也很强大。 OP 比较抓耳。

少女终末旅行

科幻 末日 日常 漫改

有导演独特个人风格的公路片。废土+履带摩托+少女，在一路上的经历。很安静的旅程，却又充满了谜团和看不到尽头的绝望/希望。

新奇诺之旅

奇幻 寓言 轻改

还没有机会补完小说和旧版动画，但是新版无论是画面还是声优都是我的菜。悠木碧是神。一集一个国家的故事，晦涩而毫不拖泥带水，就像奇诺在一个国家的短暂停留，但是看完后总会思考很久这个故事的意义。 OP 和 ED 由やなぎなぎ包办，很好听。

泥鲸之子们在沙地上歌唱

幻想 群像剧 念动力 漫改

在一个独特世界的群像剧，人物很美型，但是看到现在有些角色的性格稍微令我无法接受。设定很棒，有潜力。 ED 十分惊艳，有一定毒性。 OP 也不错。

魔法使的新娘

幻想 魔法 异种婚 漫改

目前看来故事的内容都是温馨治愈类的。除了女主的身世，还没有看到任何负面的元素。第三集尤为精彩，通过另一个物种的视角讨论了生死观。OP非常抓耳，可以评为本季最佳。

本季在追动画在中国大陆的版权完全由 Bilibili 和爱奇艺包揽，一直以来我的口味也和这两家的作品选择眼光出奇合拍。几年前在 S1 和微博有一个‘爱奇艺ACG’账号，看起来是内部人员，从他那里除了可以看到一点下季买版权的小道消息，也可以看到不少他本人对于在播动画的感想和感悟分享。然而这些年日本动画的市场逐渐向另一些人群拓展，而在我们中的一些人看来，日本动画的质量在无法忽视地下滑。‘爱奇艺ACG’大概也是离职了，这几年也不知道在以怎样的身份活动着，不知道他是不是还在像当年一样热衷于讨论新番动画的剧情和业内故事呢。

综合感觉，十月新番这几部作品的制作质量和剧情饱满程度都超出以往三年的平均水平，希望它们都能保持现在的势头，我就可以好好享受接下来的三个月了。顺便期待明年要出的有生之年的作品。
